# FafasPlugin (Paper 1.21.x)

Freunde-Server Plugin mit vielen integrierten Systemen.

## Features

- QoL: `/workbench`, `/ec`, `/anvil`
- Moderation-Light: `/mute`, `/tempban`, Chatfilter
- Join/Quit-Formatierung + Live-Scoreboard
- Economy + `/shop`, `/money`, `/pay`
- Events: Drop-Event, Kiste, Bossbar-Ankündigungen, Rotationen
- Homes/Warp: `/sethome`, `/home`, `/spawn`
- TPA: `/tpa`, `/tpaccept`, `/tpdeny`
- Teams/Party: `/party invite|accept`
- GUI-Menü: `/servermenu`
- Scheduler: Daily Reward, Rotations-Announcements
- Minigame-Welt: `/minigame join` erstellt/öffnet `fafa_minigames`

## Build

```bash
gradle clean build
```

## Essentials-Überschreibung

Die Commands sind absichtlich mit bekannten Namen registriert (`workbench`, `ec`, `home`, `spawn`, `tpa` etc.).
